# 使用

## 帮助
ncgo help

# 源码安装
go get -u github.com/wwek/ncgo

# 功能清单

- [ ] 检查网络TCP UDP端口是否打开
- [ ] httpfile, hf    基于http的文件下载和上传
- [ ✔️] speedtest, st   speedtest.net网络带宽测速 😁
- [] tcping, tcping  tcping检查某个tcp端口sync，ack时间
